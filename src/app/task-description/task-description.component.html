• Framework: Angular 18
  • UI components: Angular Material
  • CSS preprocessor: SCSS

Komponent do ustawiania hasła składa się z dwóch pól input oraz wskaźnika siły hasła i komunikatu o zgodności haseł.

  • Pola input:
    • Pole do wprowadzenia hasła.
    • Pole do powtórzenia hasła.
  • Sprawdzenie siły hasła:
    • Słabe
    • Średnie
    • Silne
    • Bardzo silne
  • Wymagania dotyczące hasła:
    • Minimum jedna duża litera
    • Minimum jedna mała litera
    • Minimum jedna cyfra
    • Minimum jeden znak specjalny
    • Minimalna długość: 10 znaków
  • Ocena siły hasła:
    • Słabe: Spełnia tylko 2 z powyższych wymagań
    • Średnie: Spełnia 3 z powyższych wymagań
    • Silne: Spełnia 4 z powyższych wymagań
    • Bardzo silne: Spełnia wszystkie wymagania
  • Wprowadzenie hasła:
    • Użytkownik wprowadza hasło w pierwszym polu.
    • Pod polem hasła wyświetlana jest siła hasła.
  • Powtórzenie hasła:
    • Użytkownik wprowadza powtórzenie hasła w drugim polu.
    • Komponent sprawdza, czy oba hasła są identyczne i wyświetla odpowiednią informację (np. "Hasła są zgodne" lub "Hasła nie są zgodne").
  • Walidacja:
    • Siła hasła jest aktualizowana na bieżąco podczas wpisywania.
    • Sprawdzanie zgodności haseł następuje po zakończeniu wpisywania w drugim polu.
  • Wskaźnik siły hasła:
    • Wizualizacja siły hasła za pomocą wskaźnika takiego jak pasek postępu
  • Informacje o zgodności haseł:
    • Wyraźne i czytelne informacje o zgodności haseł.
